## Description

## The player is introduce to a underwater world with yellow fishes floating around them where the player can use the keyboard and discover the world. The background reflects waves that change colours to envoke calmness. As the player moves right you can see islands and jellyfish and as the player moves down they can sharks. As the player moves the yellow fish follows along.

## Learning Goals

Learning Goals:

- Create world using meditative camera
- Use pacing to envoke emotion
